# Nocto Base64 (nocbase)

A fast, secure, and beautiful Base64 Encoder & Decoder built with React and Vite. It features a premium dark glassmorphism design that ensures a modern, sleek user experience.

## Features

- **Live Conversion:** Seamlessly encode and decode Base64 in real-time as you type.
- **File Upload Support:** Drag and drop or browse files up to 10MB. Easily encode files to Base64 or decode Base64 text files.
- **UTF-8 Safe:** Built with native `TextEncoder` and `TextDecoder` to safely handle emojis, special characters, and Unicode text.
- **Premium UI:** Smooth, dynamic interface using a dark glassmorphism aesthetic.
- **Quick Actions:** One-click copy to clipboard and easy download of your converted results.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` in your browser.

## Tech Stack

- [React](https://reactjs.org/) (TypeScript)
- [Vite](https://vitejs.dev/)
- Vanilla CSS (No Tailwind)
- [Lucide React](https://lucide.dev/) for icons
